# 🏁 Pitwall Manager: F1 1976

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)

Estrategia y gestión en tiempo real sin comunicación por radio. Las órdenes se transmiten al piloto mediante carteles en la línea de meta. Ambientado en la temporada 1976 de Fórmula 1 clásica.

[→ Hub de la saga](https://github.com/rreinav/pitwall-manager)

## Desarrollo

```bash
pnpm run dev     # Servidor con live reload
pnpm run lint    # Oxlint
pnpm run format  # Oxfmt
pnpm run test    # Tests (Node --test)
pnpm run deploy  # Publicar a gh-pages
```
